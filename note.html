HTML tags I've learnt so far

Paragraph tags and article tags (p , article)
Pretty self explanitory, This is a tag used to create paragraphs and articles.
Heading tags (h1 , h2, h3, etc. cant forget the /h1)
These tags are used to show headings of things like pages titles. In this case my headings are the things that provide some insight to what I will be talking about and refrencing (BIG TEXT)
Image tag (img src)
This tag is used to put an image on a HTML page make sure you have the image file in a folder when refrencing it and look theres a photo of me by using this tag
Link tag (a href)
Links used to either link to another page E.G Index -> note and or other external sites. Just like the ones I have right now on the top of the page.
List tags (ul, ol, li, dl, dt, dd)
Tags used to create lists for cool things like bullet points or just lists which is used to show this and my hobbies! (ordered or unordered).
